The following table explains all the available Table Tags in HTML



S.NO.

TAG

EXPLANATION

1. TABLE
  • Used to define a Table.
  • Useful for controlling the layout of Inforamtion.
  • Attributes : ALIGN, BGCOLOR, BORDER, CELLPADDING, CELLSPACING, HSPACE, VSPACE, VALIGN, WIDTH
2. TR
  • Used to position all items in the row of a Table.
  • Used within the Table tag.
  • Ending tag is usually omitted.
  • Attributes : ALIGN, BGCOLOR, VALIGN
3. TH
  • Used to specify the Table Headings.
  • Used within the Table tag.
  • Ending tag is usually omitted.
  • Attributes : ALIGN, BGCOLOR, COLSPAN, ROWSPAN, NOWRAP, WIDTH
4. TD
  • Used to specify entries in the row of a Table.
  • Used within the Table Row Tag of the Table tag.
  • Ending tag is usually omitted.
  • Attributes : ALIGN, BGCOLOR, COLSPAN, ROWSPAN, NOWRAP, WIDTH
5. CAPTION
  • Used to annotate a Table.
  • Attributes : ALIGN
6. ALIGN
  • Used to align the table, a row or a particular column of a row.
  • Values : CENTRE, LEFT (By Default), RIGHT
7. VALIGN
  • Used to align the table, a row or a particular column of a row vertically.
  • Values : CENTRE, TOP (By DEfault), BOTTOM
8. BGCOLOR
  • Used to provide a background color for the complete table, a row or a particular column of a row.
  • Values : Given by Name or Equivalent Hexadecimal Code.
9. BORDER
  • Used to provide a border to the table and between Rows and Columns of a Table.
  • Values : Can be given as a number, By Default - 2 Pixel
10. CELLSPACING, CELLPADDING
  • Used to control the width of the dividers between Rows and Columns and the amount of space between the Information in a cell and the Borders of the cell.
  • Values : Can be given as a number, By Default - 6 Pixel
11. COLSPAN, ROWSPAN
  • Used to span two or more columns or rows in one.
  • Values : Given as a number
12. HSAPCE, VSAPCE
  • Used to provide a space in Horizontal and Vertical direction outside the Table.
  • Values : Can be given as a number, By default - 2 Pixel
13. WIDTH
  • Used to specify the width of the complete Table.
  • Values : Can be given as a number, Can be given as percentage of the total width of the screen.
