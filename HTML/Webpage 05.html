FONT TAG


Following are the attributes of Font Tag :-

  • Face

    The Face attribute takes the name of the font to be used.
    Example :
    1. This is displayed in Arial.
    2. This is displayed in Comic Sans MS.

  • Color

    The Color attribute takes either the hexadecimal color value or just the color name.
    Example :
    1. This is displayed in Blue.
    2. This is displayed in #AF001F.

  • Size

    The Size attribute takes a number from 1 to 7 as its value with 1 as the smallest and 3 the default.
    Example :
    1. This is the smallest size of text.
    2. This is the default size of text.
    3. This is the largest size of text.
